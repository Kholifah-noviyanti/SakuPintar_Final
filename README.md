# SakuPintar - Aplikasi Manajemen Keuangan Pribadi

Aplikasi **SakuPintar** adalah proyek tugas akhir (UAS) untuk mata kuliah Pemrograman Mobile. Aplikasi ini dirancang untuk membantu pengguna mencatat transaksi pengeluaran secara dinamis dan memantau sisa saldo secara real-time.

## Fitur Utama

- **Manajemen Saldo:** Menampilkan saldo total yang berkurang otomatis setiap ada transaksi baru.
- **Input Dinamis:** Menambahkan transaksi melalui form Modal Bottom Sheet.
- **Riwayat Transaksi:** Menampilkan daftar pengeluaran menggunakan `ListView.builder` yang efisien.
- **Format Mata Uang:** Menggunakan library `intl` untuk format Rupiah (IDR) yang rapi.

## Teknologi yang Digunakan

- **Framework:** Flutter
- **Bahasa Pemrograman:** Dart
- **Library:** `intl` (Internationalization)
- **State Management:** StatefulWidget (SetState)

## Cara Menjalankan

1. Pastikan Flutter SDK sudah terinstall di perangkat Anda.
2. Clone repository ini.
3. Jalankan perintah `flutter pub get` untuk mengunduh dependencies.
4. Jalankan aplikasi dengan perintah `flutter run` atau tekan **F5** di VS Code.
